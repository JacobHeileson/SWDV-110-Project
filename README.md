This project follows the journey of the character (Named and described by the user) as they climb Idahoï¿½s tallest peak, Mount Borah. The user gets to decide what gear to bring, the routes, and how long the journey will take. This project follows object oriented programming.

Running the Program
To successfully run this program, you will need to install a Java Development Kit (JDK). Ensure the JDK has a Java Runtime Environment (JRE) and a Java compiler (javac). The file must be saved as SWDV105Projects.java, but choose a file location that is convenient to you. Open a terminal (use the Command Prompt app for Windows, or Terminal for Mac/Linux), and navigate to the location where you saved the file. Example: cd path/to/your/folder/. Then compile the program with "javac SWDV105Projects.java". You can then run the program with "java SWDV105Projects".

Usage
The usage of this program allows you to decide basic descriptions about the character (Name, Age, descriptive pronouns, i.e., he/him/his), what gear you would like them to have (water, coat, socks, trekking poles, etc.), and routes to take (i.e., turn North, keep heading East, etc.).

Features
This program features:
* System inputs and outputs (Name, Age, Gear Items, Directions to Travel)
* If statements (if, if else)
* Loops (For, While, Enhanced For)
* Arrays
* Recursion
* Polymorphism
* Story Nodes
